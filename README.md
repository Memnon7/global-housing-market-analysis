# Global Housing Market Analysis

## Project Overview
This project explores trends in the global housing market using data analysis and machine learning techniques in Python.

The analysis focuses on housing indicators such as:
- House Price Index
- Rent Index
- Affordability Ratio
- Mortgage Rates
- Inflation Rates
- GDP Growth

The project compares developed and developing economies and investigates relationships between economic indicators and housing prices.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Analysis Included
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Data visualization
- Comparison between developed and developing economies
- Linear Regression model
- Decision Tree Regression model

---

## Machine Learning Models
The project uses:
1. Linear Regression
2. Decision Tree Regressor

The models are used to predict the House Price Index based on economic indicators.

---

## Dataset
The dataset contains global housing market indicators from multiple countries between 2015 and 2024.

---

## Goal
The goal of this project is to better understand housing market trends and explore how economic factors can influence housing prices.
